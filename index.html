This keyword

A function's this references the execution

context for that call, determined entirely by

how the function was called.

*Credits: Kyle Simpson

This keyword

rules
  1. default - global object(except strict mode)
  2. call in context of an object
  3. abc - apply, bind, call
  4. new keyword

This keyword

rules

1. default - global object(except strict mode)


            function logThis () {
                console.log(this);
            }
            logThis(); // window

This keyword

rules

2. call in context of an object


            var obj = {
                logThis: function () {
                    console.log(this);
                }
            };
            obj.logThis(); // obj

This keyword

rules

3. abc - apply, bind, call


            function logThis (greeting) {
                console.log(greeting + ', ' + this);
            }
            var obj = {
                name: "John"
            };
            logThis.call(obj, "Hello"); // Hello, obj
            logThis.apply(obj, ["Hello"]); // Hello, obj
            bindedLogThis = logThis.bind(obj);
            bindedLogThis("Hello"); // Hello, obj

This keyword

rules
